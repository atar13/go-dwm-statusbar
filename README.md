# go-dwm-statusbar

Lightweight status bar for the dwm window manager. 

### Modules:
- Date
- Time
- Battery
    - Requires acpi package
- Brightness
    - Requires xbacklight (package: xorg-xbacklight)
- CPU
- RAM
- MPRIS (Media playback status)
- Pulseaudo Volume Status
    - Requires pulseaudio package
